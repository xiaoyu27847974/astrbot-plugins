# 🌤️📈 AstrBot 插件包

## 插件列表

### 1. 天气预报插件 (weather_plugin.py)
**功能**：提供实时天气信息和预报

**使用方法**：
- `天气 [城市名]` - 查询指定城市天气
- `/天气 [城市名]` - 同上

**示例**：
```
天气 北京
天气 上海
```

**输出示例**：
```
🌤️ 北京, CN 天气信息

🌡️ 温度: 22°C (体感: 25°C)
💧 湿度: 65%
📊 气压: 1013 hPa
🌪️ 风速: 3.2 m/s
👁️ 能见度: 10000 m
📝 天气: 晴朗
🕒 更新时间: 2026-04-01 11:30:00
```

### 2. 股票行情查询插件 (stock_plugin.py)
**功能**：提供实时股票行情和市场信息

**使用方法**：
- `股票 [股票代码]` - 查询指定股票行情
- `市场` - 查看主要股票行情摘要
- `/股票 [股票代码]` - 同上
- `/市场` - 同上

**示例**：
```
股票 AAPL
股票 TSLA
市场
```

**输出示例**：
```
📈 AAPL 股票行情

💰 当前价格: $175.23
🔴 涨跌: +2.45 (+1.42%)
📊 今开: $172.80
🔺 最高: $176.10
🔻 最低: $172.50
📉 昨收: $172.78
👥 成交量: 45,678,900
📅 交易日: 2026-04-01
🕒 更新时间: 2026-04-01 11:30:00
```

## 🔧 安装步骤

### 1. 配置 API 密钥
在使用插件前，需要配置相应的 API 密钥：

**天气插件**：
1. 注册 [OpenWeatherMap](https://openweathermap.org/api) 账号
2. 获取 API 密钥
3. 在 `weather_plugin.py` 中替换 `your_weather_api_key`

**股票插件**：
1. 注册 [Alpha Vantage](https://www.alphavantage.co/support/#api-key) 账号
2. 获取 API 密钥
3. 在 `stock_plugin.py` 中替换 `your_alpha_vantage_api_key`

### 2. 安装到 AstrBot
1. 将插件文件复制到 AstrBot 的插件目录
2. 重启 AstrBot 服务
3. 在 AstrBot 管理界面启用插件

### 3. 测试插件
发送以下命令测试插件是否正常工作：
```
天气 北京
股票 AAPL
市场
```
